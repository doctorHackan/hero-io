# 🦸‍♂️ Hero IO Store

A modern, responsive app store platform designed to help users discover, install, and manage their favorite mobile applications all in one place.

## 📋 Features

### Core Functionality
- **Home Page**: Hero banner with App Store and Play Store links, statistics showcase, and top 8 featured apps
- **All Apps Page**: Browse all 12+ curated apps with live search functionality, download sorting (High-Low, Low-High), and responsive grid layout
- **App Details Page**: Dynamic app information with Recharts review visualization, installation tracking, and app-not-found handling
- **My Installation**: Manage installed apps with localStorage persistence, uninstall capability, and quick access

### User Experience
- **Live Search**: Real-time app filtering by title (case-insensitive) with loading feedback
- **Download Sorting**: Filter apps by download count in ascending or descending order
- **Install Management**: Install apps with success toast notifications, track installation state in localStorage
- **Loading Animation**: Smooth page navigation with loading overlay for better UX
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop devices
- **Custom Error Pages**: 404 and app-not-found error states with recovery links

### Technical Excellence
- **Modern UI**: Clean, accessible interface built with React and responsive CSS Grid/Flexbox
- **Review Charts**: Interactive bar charts powered by Recharts to visualize app ratings
- **Toast Notifications**: Real-time user feedback for installations and uninstallations
- **Client Storage**: Persistent app installation tracking using browser localStorage
- **SPA Navigation**: Smooth route transitions with React Router v6

## 🛠️ Technologies Used

- **React 18**: User interface library with Hooks for state management
- **React Router v6**: Client-side routing and navigation
- **Vite**: Lightning-fast build tool and development server
- **Recharts**: Composable charting library for data visualization
- **React Hot Toast**: Elegant toast notifications
- **CSS3**: Custom responsive styles with Grid, Flexbox, and modern features
- **JavaScript ES6+**: Modern JavaScript with async/await and arrow functions
